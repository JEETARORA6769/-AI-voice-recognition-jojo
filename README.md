# Jojo - AI Voice Assistant

An AI-powered voice assistant built with Python that responds to voice commands.

## Features
- Play songs on YouTube
- Get live weather updates
- Voice-powered calculator
- Wikipedia search
- Tell jokes
- Hinglish responses

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python jojo.py`
3. Say "Jojo" followed by your command

## Commands
| Command | Action |
|---------|--------|
| "Jojo play [song]" | Plays a song on YouTube |
| "Jojo weather" | Gets live weather |
| "Jojo calculate" | Opens calculator |
| "Jojo who is [person]" | Wikipedia lookup |
| "Jojo joke" | Tells a joke |
| "Jojo time" | Current time |
| "Jojo stop" | Exits the assistant |
